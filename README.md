# OTT Platform (Stream24)

Full-stack OTT platform built as a monorepo.

### Structure
\\\
stream24/
  frontend/   ReactJS + TailwindCSS
  backend/    Spring Boot (or Node)
\\\

### Commands
To run frontend:
\\\
cd stream24/frontend
npm install
npm start
\\\

To run backend:
\\\
cd stream24/backend
# Depending on setup, run your Spring Boot or Node server
\\\
